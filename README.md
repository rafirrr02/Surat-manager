# 📄 Surat Manager - UAS PBO

Aplikasi desktop Java untuk manajemen surat masuk dan surat keluar.

## Fitur

- Surat Masuk & Keluar dengan 10 data dummy
- Tambah, Edit, Hapus surat
- Pencarian berdasarkan nomor surat, perihal, pengirim/penerima
- Filter berdasarkan jenis (semua / masuk / keluar)
- Sorting berdasarkan tanggal (terbaru / terlama)
- Export laporan ke HTML (bisa print to PDF dari browser)

## Cara Menjalankan

1. Install Java JDK 11 atau lebih baru
2. Compile: `javac src/*.java -d out`
3. Jalankan: `java -cp out MainApp`
